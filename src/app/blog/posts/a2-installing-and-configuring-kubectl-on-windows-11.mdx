## Introduction

The kubectl command-line tool allows you to run commands against Kubernetes clusters, making it an essential tool for Kubernetes administrators and developers. In this guide, we’ll walk through downloading, installing, and configuring kubectl on Windows 11.

- Official doc: [Install and Set Up kubectl on Windows](https://kubernetes.io/docs/tasks/tools/install-kubectl-windows/#install-kubectl-binary-on-windows-via-direct-download-or-curl)

---

## Step 1: Download the Latest kubectl Version

- Visit the [official Kubernetes releases page for kubectl](https://kubernetes.io/docs/tasks/tools/).
- Download the kubectl binary for Windows by following the download link provided on the page.

If you have curl installed, use this command:

### Step 2: Move kubectl to Your System Path (Optional)

To make `kubectl` accessible from any command prompt window, add it to your system’s `PATH`.

- **Option 1**: Move the `kubectl.exe` file to a folder already included in your system’s `PATH`.
- **Option 2**: Manually Add kubectl to the PATH

1. **Open Environment Variables:**
   - In the Windows search bar, type **"Environment Variables"** and select **Edit the system environment variables**.

2. **Edit the PATH Variable:**
   - Under **System Variables**, find the **Path** variable and click **Edit**.
   - Add the folder path where `kubectl.exe` is saved (e.g., `C:\kubectl`).

---

### Step 3: Verifying kubectl Installation

Open Command Prompt or PowerShell and enter the following command to verify the installation:

If installed correctly, you’ll see version information for the kubectl client, confirming that it’s ready to use.

Expected output:

```
> C:\Users\a848733>kubectl version --client
> Client Version: v1.31.0
> Kustomize Version: v5.4.2
```

In this post, we covered the steps to install and configure kubectl on Windows 11. With kubectl installed, you’re ready to manage your Kubernetes clusters directly from the command line.
